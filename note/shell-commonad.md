
### github命令

```shell
git cherry-pick <commit_id>
```

挑出commit历史的某一个commit，变成最新的commit提交


```shell
git pull --rebase
```
在本地master或main分支输入命令，可拉取到本地并rebase到本地



```shell
git checkout <branch-name>
```
切换到对应分支



```shell
git rebase main
```
```shell
git rebase master
```
开发分支项目时，把本地的主分支更新到当前分支
